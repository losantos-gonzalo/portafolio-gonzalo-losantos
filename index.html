Sobre mi

Soy un desarrollador web con sólidos conocimientos en HTML, CSS, Sass, Git y Bootstrap. Mi pasión radica en la creación de sitios web atractivos y funcionales, así como en la optimización de su rendimiento para brindar la mejor experiencia posible al usuario.

Con habilidades en maquetación de páginas web, puedo convertir diseños en código limpio y estructurado utilizando HTML y CSS. Mi enfoque se basa en la creación de sitios web intuitivos y fáciles de navegar, asegurando una experiencia agradable para los visitantes.

Además, utilizo Sass como preprocesador de CSS para mantener un código modular, reutilizable y fácil de mantener. Esto me permite ahorrar tiempo y esfuerzo al desarrollar estilos personalizados y garantizar la consistencia en todo el sitio web.

Mi experiencia en Git y GitHub me permite gestionar eficientemente el control de versiones de mis proyectos y colaborar de manera efectiva con otros desarrolladores. Estoy familiarizado con la creación de repositorios, la gestión de ramas y la fusión de cambios, lo que facilita el trabajo en equipo y la entrega de proyectos de calidad.

Además, cuento con habilidades en el uso de Bootstrap, un framework CSS popular y ampliamente utilizado. Con Bootstrap, puedo crear diseños responsivos y adaptativos de manera rápida y eficiente, lo que garantiza que los sitios web se vean y funcionen bien en diferentes dispositivos y tamaños de pantalla.
